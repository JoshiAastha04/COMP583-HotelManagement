Some simple Git Rules:-

Do** NOT push** anything directly to **main**.

Always **create a new branch** before working.

After finishing your work, create a **Pull Request to merge into main**. 


**Setting Up React Project** (WebStorm) 

We are using **Vite + React**.

Prerequisites :- Make sure **Node.js is installed**:

node -v
npm -v

Create the Project In WebStorm terminal, run:

npm create vite@latest


Enter** project name, 
Select React, 
Select JavaScript
**

Then:

cd project-name
npm install
npm run dev


Open the local URL shown in the terminal (usually http://localhost:5173).
